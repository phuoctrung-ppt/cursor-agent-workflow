# Growth Loop

## Weekly loop
1. Ship at least one meaningful workflow improvement.
2. Publish one public update post (X/LinkedIn/Facebook/Dev community).
3. Include one real use case + one measurable result.
4. Ask for feedback from 5–10 users and capture testimonials.

## Per-release checklist
- Update `CHANGELOG.md`
- Share "what changed" + "who should care"
- Add one screenshot/GIF or short demo video
- Open or update roadmap issue status
