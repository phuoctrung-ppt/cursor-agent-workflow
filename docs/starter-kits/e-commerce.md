# Starter Kit: E-commerce

## Use this when
- You are building cart/checkout/order workflows and admin operations.

## First customizations
1. Define order/payment/inventory compliance rules in `AGENTS.md` §6.
2. Configure protected paths for payment/auth/inventory critical files.
3. Split module plans by domain: catalog, cart, checkout, fulfillment.
4. Enforce judge reviews for pricing and order-state changes.

## Suggested first module
- Catalog + cart basics with acceptance criteria in `docs/user-stories/`.
