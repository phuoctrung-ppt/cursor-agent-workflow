# Starter Kit: SaaS

## Use this when
- You are building multi-module SaaS with API + web + auth.

## First customizations
1. Update `AGENTS.md` §1–§3 with your actual product and repo layout.
2. Define auth/RBAC/compliance in `AGENTS.md` §6.
3. Tighten worker scopes in `.cursor/config/worker-scopes.json`.
4. Add sensitive paths in `.cursor/config/protected-paths.json`.

## Suggested first module
- Authentication + tenant/workspace setup.
