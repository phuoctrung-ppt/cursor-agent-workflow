# Migration Guide: From Scratch

1. Copy `.cursor/` and `AGENTS.md` into the new repository.
2. Fill all placeholders in `AGENTS.md`.
3. Run `/architecture-plan brainstorming "<product idea>"`.
4. Run `/architecture-plan` to generate executable tasks.
5. Start with `/dev-module <first-module>`.
6. Verify artifacts appear in `docs/plans/`, `docs/adr/`, and `docs/reviews/`.
